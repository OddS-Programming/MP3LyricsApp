## 📋 Описание

**MP3 Lyrics Metadata Editor** — это мощное консольное приложение на Java, которое позволяет:

- ✏️ **Редактировать метаданные MP3** (название, артист, альбом, год, жанр и т.д.)
- 🔍 **Автоматически искать тексты песен** на сайте Genius
- 📝 **Добавлять полные тексты** прямо в метаданные MP3
- 💫 **Работать с полным текстом** без обрезок и ограничений

Приложение использует **Selenium WebDriver** для автоматизации браузера Chrome, что гарантирует надёжный поиск текстов без блокировок и ограничений API.

### ✨ Ключевые особенности

```
✅ Полная поддержка ID3 тегов MP3
✅ Автоматический поиск на Genius через Chrome
✅ Вывод полного текста песни без обрезок
✅ Простой и интуитивный консольный интерфейс
✅ Безопасность: изменения только по подтверждению
✅ Поддержка Windows, Linux, macOS
✅ Автоматическое управление ChromeDriver
```

### Детальное описание функций

#### 1. 📂 Загрузить MP3 файл

Загружает MP3 файл в памяти приложения для последующего редактирования.

**Входные данные:** Полный путь к MP3 файлу  
**Вывод:** Подтверждение успешной загрузки  
**Примеры путей:**
- Windows: `C:\Users\Music\song.mp3`
- Linux/Mac: `/home/user/Music/song.mp3`

```
📂 Введи путь к MP3 файлу: C:\Music\TheWeeknd-BlindingLights.mp3
✅ Файл загружен успешно: TheWeeknd-BlindingLights.mp3
```

#### 2. 📖 Посмотреть метаданные

Отображает все текущие метаданные загруженного MP3 файла.

**Доступные поля:**
- 🎵 Название (Title)
- 🎤 Артист (Artist)
- 💿 Альбом (Album)
- 📅 Год (Year)
- 🎼 Жанр (Genre)
- 👥 Артист альбома (Album Artist)

```
📋 ТЕКУЩИЕ МЕТАДАННЫЕ:
───────────────────────────────────────
🎵 Название: Blinding Lights
🎤 Артист: The Weeknd
💿 Альбом: After Hours
📅 Год: 2019
🎼 Жанр: Synthwave
👥 Артист альбома: The Weeknd
───────────────────────────────────────
```

#### 3. ✏️ Редактировать метаданные

Интерактивное редактирование каждого поля метаданных.

**Процесс:**
- Введи новое значение для каждого поля
- Оставь пустым для пропуска
- Подтверди каждое изменение нажатием Enter

```
🎵 Новое название (пусто - пропустить): Starboy
🎤 Новый артист (пусто - пропустить): The Weeknd ft. Daft Punk
💿 Новый альбом (пусто - пропустить): Starboy
📅 Новый год (пусто - пропустить): 2016
🎼 Новый жанр (пусто - пропустить): Electro House

✅ Метаданные обновлены. Нажми '5' для сохранения!
```

#### 4. 🔍 Поиск текстов на Genius

Автоматический поиск и загрузка полного текста песни с сайта Genius.

**Процесс работы:**
1. Вводишь имя артиста
2. Вводишь название песни
3. Приложение формирует URL: `https://genius.com/artist-track-lyrics`
4. Открывает Chrome и загружает страницу
5. Ждёт полной загрузки текста через JavaScript
6. Парсит и выводит полный текст

```
🎤 Введи имя артиста: The Weeknd
🎵 Введи название песни: Blinding Lights
🔍 Ищу текст на Genius...
   📡 Запускаю Chrome...
   🔗 Открываю: https://genius.com/the-weeknd-blinding-lights-lyrics
   📝 Извлекаю текст...
   ✅ Текст загружен (2847 символов)

✅ НАЙДЕНО!
───────────────────────────────────────
🎵 Песня: Blinding Lights
🎤 Артист: The Weeknd
───────────────────────────────────────

📝 ПОЛНЫЙ ТЕКСТ ПЕСНИ:

I can't sleep until I feel your touch
And every night this is hard to swallow
But what else can I do
When I'm not with you I'm paralyzed
[... весь текст без обрезок ...]

💾 Добавить полный текст в метаданные? (y/n): y
✅ Текст добавлен в метаданные!
```

#### 5. 💾 Сохранить изменения

Сохраняет все изменения метаданных в MP3 файл.

```
💾 Выбери опцию (1-6): 5
✅ Метаданные сохранены успешно!
📁 Файл: C:\Music\TheWeeknd-BlindingLights.mp3
```

## 🐛 Решение проблем

### ❌ Ошибка: "Chrome not found"

**Проблема:** Приложение не находит установленный Chrome

**Решения:**
1. Убедись, что Google Chrome установлен
2. Проверь версию Chrome: `chrome://version/`
3. Переустанови Chrome или Chromium

```bash
# Windows
choco install googlechrome

# macOS
brew install google-chrome

# Linux Ubuntu
sudo apt-get install google-chrome-stable
```

### ❌ Ошибка: "MP3 file not found"

**Проблема:** Файл не найден по указанному пути

**Решения:**
- Проверь правильность пути к файлу
- Убедись, что файл имеет расширение .mp3
- Используй абсолютный путь: `C:\Users\Music\song.mp3`
- Избегай пробелов в названии папок

### ❌ Ошибка: "Text not found on Genius"

**Проблема:** Текст песни не найден на Genius

**Решения:**
1. Проверь правильность написания названия артиста и трека
2. Убедись, что песня есть на Genius.com
3. Попробуй другой артист/трек
4. Проверь интернет-соединение

### ⚠️ Предупреждение: "SLF4J: No providers"

Это не ошибка — просто предупреждение о логировании. Приложение будет работать нормально.
